Notes de version

Consultez le journal des modifications pour toutes les versions sur le wiki GitHub

Nouveautés dans la version 1.3.5

  • Correction : le texte pouvait s'afficher à des tailles différentes d'une plateforme à l'autre, même avec la même police et taille de police. A partir de cette version, tout document créé ou modifié devrait s'afficher identiquement sur toutes les plateformes (tant qu'il est ouvert avec OpenBoard 1.3.5 minimum)
  • Correction : en mode document, la suppression d'un document par drag-n-drop pouvait faire planter OpenBoard
  • Correction : les éléments audio/vidéo ne pouvaient pas être sélectionnés par un rectangle de sélection
  • Mode document: possibilité de déplacer une sélection multiple
  • Ajout d'un paramètre dans les fichiers de configuration pour la couleur du quadrillage de fond de page *
  • Ajout de la traduction Basque (crédit: Inigo Gebara)
  • Mise à jour de la traduction Allemande (crédit: EmPeWe)
  • Couleur du cercle du marqueur corrigé en mode bureau
  • Correction : En mode bureau, avec le mode multi-écran désactivé, l'écran était parfois tout de même cloné
  • Correction : en mode document, OpenBoard pouvait planter lors de la suppression de pages d'un document
  • Correction : les groupes d'objets et les traits ne conservaient pas leur verrouillage à l'enregistrement du document
  • Correction : les groupes d'objets ne conservaient pas leurs transformations (déplacement/rotation) à l'enregistrement du document
  • Ajout d'un rayon minimum pour le rapporteur, pour qu'il ne puisse pas être redimensionné jusqu'à être inutilisable
  • Taille du compas diminuée légèrement
  • Taille minimum de l'équerre diminuée
  • Correction : un groupe déplacé en cliquant dessus directement (pas sur le cadre) perdait son déplacement à l'enregistrement
  • Correction : les objets verrouillés pouvaient être déplacés (en cliquant dessus directement, pas en déplaçant le cadre)
  • Correction : La position des groupes de traits était mal enregistrée
  • Correction : sur Linux et Windows, sur des écrans à basse résolution, les boutons des boîtes de texte pouvaient dépasser de leur cadre
  • Correction : crash à la fermeture du logiciel avec une page vide, mais modifiée (p. ex si un élément avait été ajouté à la page puis supprimé)
  • Correction : sur Windows, l'importeur Open-Sankoré ne se lançait pas

(*) Ceci permet à l'utilisateur de changer la couleur ou l'opacité du quadrillage, par exemple s'il n'est pas assez visible sur un écran à bas contraste ou un projecteur. Les paramètres se trouvent sous la catégorie Board et sont nommés CrossColorDarkBackground et CrossColorLightBackground pour les fonds foncés et clairs, respectivement. Les valeurs possibles sont des chaînes de caractères dans n'importe lequel des formats suivants:

  • #RGB (Hexadécimal)
  • #RRGGBB
  • #AARRGGBB
  • #RRRGGGBBB
  • #RRRRGGGGBBBB
  • Mots-clés de couleur SVG (tels que définis par W3C)

N.B: A = alpha (transparence); R = rouge; G = vert; B = bleu

Les valeurs par défaut sont :

CrossColorDarkBackground=#C82C2C2C
CrossColorLightBackground=#A5E1FF
