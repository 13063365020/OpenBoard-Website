What is OpenBoard ?

OpenBoard is an open source cross-platform interactive whiteboard application designed primarily for use in schools and universities. It can be used both with interactive whiteboards or in a dual-screen setup with a pen-tablet display and a beamer.

OpenBoard was originally forked from Open-Sankoré, which was itself based on Uniboard.

Main features include

  • Drawing tools (pen, highlighter)
  • Many different elements can be added to the board, including text boxes, pictures, videos, music and widgets (native or HTML/javascript)
  • Easy management of all these elements through a built-in "library" pane
  • Search tools to retrieve pictures, videos and audio from Planète Sankoré, abulEdu.org and Pixabay.com
  • Screencast function (OS X and Windows only)
  • Desktop mode, to easily annotate anything outside of OpenBoard
  • Integrated web browser
